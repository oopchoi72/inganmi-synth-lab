# Persona Cards

## 1) 완벽주의형
- 목표: 높은 완성도, 실수 최소화
- 스트레스: 불확실한 문구, 되돌리기 불가
- 리뷰 포인트: 검토/수정/되돌리기 동선

## 2) 벼락치기형
- 목표: 짧은 시간 내 완료
- 스트레스: 단계 과다, 불필요한 확인
- 리뷰 포인트: 최소 클릭으로 완료 가능성

## 3) 습관 루틴형
- 목표: 매일 짧게 꾸준히 기록
- 스트레스: 반복 입력 피로, 흐름 끊김
- 리뷰 포인트: 연속성/재진입 편의

## 4) 불안·회피형
- 목표: 부담 없이 시작
- 스트레스: 평가받는 느낌, 실패 메시지
- 리뷰 포인트: 진입 장벽/회복 경로

## 5) 파워유저형
- 목표: 빠른 조작, 고급 제어
- 스트레스: 느린 플로우, 장황한 가이드
- 리뷰 포인트: 우회 경로/단축 동선

## 6) 사회적 동기형
- 목표: 피드백/연결 기반 동기 유지
- 스트레스: 공유 불안, 반응 부재
- 리뷰 포인트: 공유/피드백 흐름 명확성
